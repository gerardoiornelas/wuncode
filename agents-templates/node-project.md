## Project Overview
- Node.js or frontend project using repository-local scripts.

## Repository Structure
- Respect the existing app and package boundaries.
- Reuse utilities before adding new helpers.

## Coding Standards
- Follow the repo formatter and linter.
- Keep component logic readable and testable.

## Test Commands
- `npm run test`
- `npm run lint`

## Commit Checklist
- Run targeted tests.
- Avoid unrelated formatting churn.
- Keep commits focused.

## Local Model Constraints
- Prefer small, relevant file sets.
- Summarize large generated files instead of pasting them.
